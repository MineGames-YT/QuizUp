# QuizUp

QuizUp --- это веб-игра для проведения командной викторины в реальном
времени.\
Две команды соревнуются между собой, отвечая на вопросы по выбранной
теме.\
Вопросы могут генерироваться через Groq (Llama 3) или использоваться из
запасного набора.

Проект реализован на Flask + WebSocket (Flask-SocketIO).

## Возможности

-   Создание комнаты с уникальным 6-значным PIN-кодом\
-   Вход игроков по PIN\
-   Автоматическое разделение на 2 команды\
-   Лобби со списком подключённых игроков\
-   Запуск игры ведущим\
-   Поочерёдные вопросы командам\
-   Таймер на каждый вопрос\
-   Подсчёт очков в реальном времени\
-   Определение победителя\
-   Генерация вопросов через Groq API\
-   Автоматический fallback на локальные вопросы при ошибке API\
-   Кастомные уведомления (без браузерных alert)\
-   Светлая и тёмная тема

## Технологии

Backend: - Python 3.10+ - Flask - Flask-SocketIO - SQLAlchemy - Eventlet

Frontend: - HTML - CSS - JavaScript - Socket.IO client

AI: - Groq API (OpenAI-совместимый формат) - Модель по умолчанию:
llama-3.3-70b-versatile

## Установка

Склонировать репозиторий или распаковать архив:

git clone `<repo_url>`{=html}\
cd quizup

Создать виртуальное окружение:

python -m venv venv\
source venv/bin/activate

Установить зависимости:

pip install -r requirements.txt

## Настройка .env

Создать файл `.env` в корне проекта:

GROQ_API_KEY=ваш_ключ\
GROQ_MODEL=llama-3.3-70b-versatile\
GROQ_API_BASE_URL=https://api.groq.com/openai/v1\
GROQ_ENABLED=1

Если нужно временно отключить генерацию через Groq:

GROQ_ENABLED=0

В этом случае будут использоваться локальные вопросы.

## Запуск

python app.py

После запуска приложение будет доступно по адресу:

http://127.0.0.1:5000

## База данных

Используется SQLite.\
Файл базы создаётся автоматически в папке instance/.\
Ничего вручную создавать не требуется.

## Если Groq не работает

Возможные причины: - неверный API-ключ - превышен лимит - отсутствует
баланс - ошибка сети

В этом случае: - игра не падает - автоматически используются запасные
вопросы - выводится кастомное уведомление
